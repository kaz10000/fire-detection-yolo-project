Collection Rules / 수집 규칙

1. 데이터 출처는 반드시 기록합니다.
2. 데이터 수집 개수를 클래스별로 기록합니다.
3. 큰 데이터셋은 GitHub에 직접 올리지 않고 링크로 공유합니다.
4. 실제 화재를 연출하여 촬영하지 않습니다.
5. 저작권이나 사용 조건이 불분명한 데이터는 사용하지 않는 것이 좋습니다.
6. 학습에 사용한 데이터와 테스트에 사용한 데이터를 구분합니다.


1. Always record the source of the dataset.
2. Record the number of images for each class.
3. Do not upload large datasets directly to GitHub; share them using links.
4. Do not create real fire scenes for filming.
5. Avoid data with unclear copyright or usage conditions.
6. Separate training data and testing data.
