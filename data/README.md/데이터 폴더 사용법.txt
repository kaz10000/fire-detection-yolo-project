# Data Folder Guide

## Folder Structure

raw/
- Original collected images or videos

processed/
- YOLO training dataset

## Classes
- fire
- smoke
- normal/background

## Rules
1. Store original data in `raw/`.
2. Store YOLO-ready data in `processed/`.
3. Record every dataset source in `dataset_sources.md`.
4. Do not upload very large files directly to GitHub.
5. Use Google Drive or Roboflow for large datasets.

## Recommended Dataset Size
- fire: 500 images
- smoke: 500 images
- normal/background: 500 images
- total: about 1,500 images
