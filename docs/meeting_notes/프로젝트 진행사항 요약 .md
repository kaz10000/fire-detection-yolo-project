| 평가 지표 | 결과 |
|---|---:|
| Precision | 약 97.8% |
| Recall | 약 96.9% |

## 현재 확인된 문제점

| 문제 | 설명 | 개선 방향 |
|---|---|---|
| 약한 불꽃 감지 불안정 | 작은 불꽃이나 흐릿한 불꽃을 잘 감지하지 못함 | 약한 불꽃 이미지 추가 학습 |
| 야외 연기 감지 부족 | 산불, 드론 시점, 넓게 퍼진 연기 감지가 약함 | wildfire smoke 데이터 추가 |
| 테스트 영상과 학습 데이터 차이 | 학습 데이터는 가까운 화재 이미지가 많고, 테스트 영상은 야외/원거리 장면이 포함됨 | 테스트 영상 프레임 추출 후 재학습 |
| 영상 프레임별 감지 흔들림 | 한 프레임에서는 감지되고 다음 프레임에서는 감지가 사라질 수 있음 | 경고 유지 시간 적용 |

## 발표에 사용할 핵심 문장

기존 제안서에서는 여러 위험 상황 감지를 목표로 하였으나, 프로젝트 기간과 구현 완성도를 고려하여 본 프로젝트의 1차 구현 범위를 화재 감지로 한정하였다.  
YOLOv8n 모델을 활용하여 화재 및 연기 객체를 탐지하고, 위험 상황이 감지되면 한글 UI 화면에 경고 메시지를 표시하도록 구현하였다.

## English Summary

This project originally proposed fire detection, fall detection, and dangerous object detection.  
However, based on the professor’s feedback and project feasibility, the current implementation focuses on **fire detection**.

The YOLOv8n model was trained using a Roboflow Fire-Smoke dataset in Kaggle Notebook.  
The trained model achieved high validation performance, but some limitations were found in weak flames, outdoor smoke, and drone-view fire videos.

Future improvement will require additional training data such as wildfire smoke, weak flame images, and aerial-view fire scenes.
