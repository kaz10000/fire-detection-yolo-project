# Training Parameters

> Kaggle Notebook에서 YOLOv8n 모델을 학습할 때 실제로 사용한 파라미터 정리

---

## Model Information

| Item | Value | 설명 |
|---|---|---|
| Model | YOLOv8n | 사용한 YOLO 모델 종류 |
| Training Method | Transfer Learning | 기존에 학습된 모델을 가져와 우리 데이터셋에 맞게 추가 학습하는 방식 |
| Base Weight | `yolov8n.pt` | YOLOv8n 사전학습 가중치 파일 |
| Final Model | `best.pt` | 학습 결과 중 가장 성능이 좋았던 최종 모델 파일 |
| Training Environment | Kaggle Notebook | 모델 학습을 진행한 클라우드 환경 |
| Dataset | Roboflow Fire-Smoke Detection Dataset | 학습에 사용한 화재·연기 데이터셋 |

---

## Actual Training Parameters

| Parameter | Value | 한글 설명 |
|---|---:|---|
| Image Size | 640 × 640 | 학습할 때 입력 이미지를 640×640 크기로 맞춤 |
| Initial Epochs | 20 | 처음 학습을 20번 반복하도록 설정 |
| Resumed Epochs | 10 | 중간에 학습이 끊긴 후 `last.pt`를 기준으로 추가 10번 학습 |
| Batch Size | 4 | 한 번에 4장씩 이미지를 묶어서 학습 |
| Patience | 3 | 성능 개선이 3번 동안 없으면 조기 종료하는 설정 |
| Cache | False | 데이터셋을 메모리에 저장하지 않고 학습하여 메모리 사용량을 줄임 |
| Workers | 1 | 데이터 로딩 작업을 1개 프로세스로 제한하여 안정성 확보 |
| Device | GPU | Kaggle의 GPU를 사용하여 학습 속도를 높임 |
| Demo Confidence Threshold | 0.15 | 시연 화면에서 15% 이상 확신하면 감지 결과로 표시 |

---

## Parameter Explanation

| Parameter | Meaning |
|---|---|
| `epochs` | 전체 데이터셋을 몇 번 반복해서 학습할지 정하는 값 |
| `imgsz` | YOLO 모델에 입력되는 이미지 크기 |
| `batch` | 한 번에 학습하는 이미지 개수 |
| `patience` | 성능 개선이 없을 때 몇 epoch까지 기다릴지 정하는 값 |
| `cache` | 데이터를 메모리에 저장해서 빠르게 학습할지 여부 |
| `workers` | 데이터를 불러오는 작업자 수 |
| `device` | CPU 또는 GPU 중 어떤 장치로 학습할지 설정 |
| `conf` | 객체를 감지했다고 판단할 최소 신뢰도 기준 |

---

## Training Code

```python
model.train(
    data=f"{DATASET_PATH}/data.yaml",  # 학습 데이터 위치와 클래스 정보가 들어 있는 설정 파일
    epochs=10,                         # 전체 데이터셋을 10번 반복 학습
    imgsz=640,                         # 입력 이미지를 640×640 크기로 사용
    batch=4,                           # 한 번에 4장씩 묶어서 학습
    patience=3,                        # 3번 동안 성능 개선이 없으면 조기 종료
    cache=False,                       # 메모리 부족 방지를 위해 캐시 사용 안 함
    workers=1,                         # 안정적인 학습을 위해 데이터 로딩 작업자 수를 1로 설정
    device=0,                          # GPU 0번 장치를 사용
    name="fire_detection_yolov8n_resume_safe"  # 학습 결과 저장 폴더 이름
)
